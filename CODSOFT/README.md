# CODSOFT Python Programming Internship

This repository contains the tasks completed during the Python Programming virtual internship at CodSoft. All projects are developed using Python and feature a modern Graphical User Interface (GUI) built with `tkinter`.

## 📚 Tasks Included

Here are the details of the three mandatory tasks completed for this internship:

### 1. Task 1: To-Do List Application
[span_1](start_span)A real-time GUI application that helps users manage and organize their daily tasks efficiently[span_1](end_span). 
* **Features:** Add new tasks, delete completed tasks, and clear the entire list.
* **Concepts Used:** Data structures, GUI layout management, event handling.

### 2. Task 2: Simple Calculator
[span_2](start_span)A basic arithmetic calculator designed with a user-friendly interface[span_2](end_span).
* **Features:** Performs addition, subtraction, multiplication, and division. [span_3](start_span)Includes error handling for zero division[span_3](end_span).
* **Concepts Used:** Mathematical operations, string evaluation (`eval`), exception handling.

### 3. Task 3: Secure Password Generator
[span_4](start_span)A utility tool that generates strong, random passwords based on user-defined length[span_4](end_span).
* **[span_5](start_span)Features:** Combines letters, numbers, and symbols to ensure password complexity[span_5](end_span). [span_6](start_span)Displays the generated password directly on the screen[span_6](end_span).
* **Concepts Used:** `random` and `string` modules, input validation.
